# E-Commerce-Performance-Profitability-Intelligence

​This project provides an end-to-end data analytics solution for a retail e-commerce business. I managed the entire data lifecycle—from raw data ingestion and cleaning in Python, through deep-dive database querying in SQL, to final executive visualization in Power BI.

Key Technical Contributions
​Data Engineering (Python/Pandas): Cleaned and standardized 3,500+ sales records.
​Resolved data type inconsistencies (Date-Time objects).
​Engineered new features: Profit_Margin_Pct and Month for time-series analysis.
​Database Analytics (SQL): Built a relational structure to perform regional performance audits.
​Utilized Aggregations and Subqueries to identify that the West Region generated the highest revenue ($2.8M).
​Calculated a consistent 17% profit margin across all geographic sectors.
​Business Intelligence (Power BI): Designed a high-impact dashboard with interactive slicers.
​Visualized sales trends by category (Electronics leading at ~50%).
​Implemented drill-down features to allow stakeholders to view regional efficiency instantly.
​Business Impact
​Identified a 15% revenue gap between the North and West regions, providing data-driven evidence to support regional marketing budget reallocation.
